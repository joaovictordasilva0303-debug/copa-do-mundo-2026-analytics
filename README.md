# Copa do Mundo 2026 Analytics

Projeto de engenharia e análise de dados desenvolvido em Python para coletar, armazenar e analisar informações da Copa do Mundo FIFA 2026.

## Objetivo

Construir uma base de dados estruturada a partir de uma API esportiva, permitindo análises estatísticas e exploração de dados das partidas.

## Tecnologias

* Python
* SQLite
* Requests
* Pandas
* Jupyter Notebook
* Git & GitHub

## Funcionalidades

* Coleta automática de partidas da Copa do Mundo 2026
* Armazenamento em banco SQLite
* Atualização incremental dos dados
* Consultas exploratórias em notebook
* Estrutura preparada para dashboards e análises avançadas

## Estrutura do Projeto

* criar_banco.py → criação do banco de dados
* atualizar_banco.py → atualização dos dados
* extrair_partida.py → extração via API
* consultar.ipynb → consultas e análises
